Coach Hub (Preview)

This is a public preview of the future Coach Hub. The live version will be a private app for verified coaches only.

[ Coach / badge collage or practice room photo ]

What Coach Hub is for

  • A. Roster management and athlete XP review across Foundry 8 and Foundry 4.
  • B. Ceremony planning — eligibility checks, scripts, and printable materials.
  • C. Standards board updates, announcements, and system changes for staff.
  • D. Coach resources: drill libraries, PDF binders, and ready-to-run practice templates.
How it will work (tech)

Coach Hub will live inside a private dashboard protected by Firebase Auth. Only authenticated coaches will be able to view rosters and XP. Data will be stored in Firestore with Security Rules guarding every read and write.

Roadmap
  • Phase 1: Public preview (this page) to show intent and structure.
  • Phase 2: Private app shell with secure login and basic coach views.
  • Phase 3: Live roster tools, ceremony planner, and shared resources.
