# Sandman Sidekick™ — Changelog

Official version history of the Sandman System™ AI Sidekick.  
Every entry ties to a phase milestone so progress is tracked forever.

---

## [v0.1.0] — Phase 38 (2025-09-01)
- **Type:** Placeholder release
- **Features:**
  - Added ON/OFF toggle for demo control
  - Status badge (green = ON, red = OFF)
  - Placeholder notes, weekly summary, and insights
  - Version label in top-right corner
- **Purpose:** First placeholder for Sandman Sidekick™, showing where the AI layer will live inside the system.

---

## [v0.2.0] — Planned Phase 45
- **Type:** Live Data Prototype
- **Features (Planned):**
  - Pulls real XP totals from Firestore
  - Highlights leader and closest competitor dynamically
  - Displays weekly summary from live practice/tournament logs
  - Alerts for duplicates or missing logs
- **Purpose:** First step toward an “AI Coach” pulling insights directly from the athlete database.

---

## [v0.3.0] — Planned Phase 55
- **Type:** Insights Expansion
- **Features (Planned):**
  - AI-generated tips (“Athlete X has low recovery XP, consider rest.”)
  - Detects streaks (practice streaks, attendance streaks, etc.)
  - Personalized encouragement snippets
- **Purpose:** Move from raw data to real **coaching intelligence**.

---

## [v1.0.0] — Future Release (Post Phase 85)
- **Type:** Full AI Coach Integration
- **Features (Planned):**
  - Sandman Sidekick™ acts as a pocket AI coach
  - Summarizes XP, recovery, challenges, and skill gaps
  - Predictive alerts: burnout risks, leader shifts, at-risk athletes
  - Custom athlete recommendations
- **Purpose:** Complete prestige AI layer, fully integrated with Sandman System™.

---

## Versioning Rules
- All Sidekick builds follow **semantic versioning** (`MAJOR.MINOR.PATCH`).
- **MAJOR** = Big system leap (ex: placeholder → full AI coach).
- **MINOR** = New features, new insights, live data layers.
- **PATCH** = Small tweaks or fixes.
