Coach Hub

Coach Hub is the operating console for coaches running Sandman Combat™ programs across Foundry 8 and Foundry 4. It becomes live only inside the private coach login.

Status: Preview Only · Coach-First
[ Coach / badge collage or practice room photo ]

What Coach Hub is for

  • Roster + athlete status — Tier, Rank, readiness, notes (Foundry 8 + Foundry 4).
  • Daily ops — practice plan, checklists, quick tools that keep sessions clean.
  • Ceremony readiness — eligibility checks, scripts, and printable materials.
  • Staff comms — standards updates, announcements, and “what changed” guidance.
What Coach Hub is NOT

Coach Hub is not the resource library. PDFs, handouts, and system explanations live in Coach Resources.

Roadmap
  • Phase 1: Public preview (this page) to show intent and structure.
  • Phase 2: Private coach login + basic shell.
  • Phase 3: Live roster tools, daily ops, and ceremony planner.
