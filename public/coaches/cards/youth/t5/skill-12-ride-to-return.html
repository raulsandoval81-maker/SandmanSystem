Skill 12 — Ride to Return

Tree: Top

Tier: Youth T5 — Champion

Structure (A/B/C)

A — Ride Control

B — Lift and Return

C — Re-Establish Ride

Coaching Cues

  • “Stay tight on top”
  • “Return with control”
  • “Land ready to ride again”

Common Mistakes

  • Losing control during return
  • Standing too tall
  • Not re-securing position

Drill Ideas

  • Ride to return reps
  • Return cycles
  • Top reset rounds

Progression

  • Connect to escape chain
