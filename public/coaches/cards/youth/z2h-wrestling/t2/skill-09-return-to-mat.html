Skill 9 — Return to Mat

Tree: Top

Tier: Youth T2 — Combatant

Structure (A/B/C)

A — Lift Control

B — Return Safely

C — Land in Position

Coaching Cues

  • “Stay tight”
  • “Bring him back under control”
  • “Land ready”

Common Mistakes

  • Losing control during return
  • Standing too tall
  • Landing without position

Drill Ideas

  • Return reps
  • Stand-up return drills
  • Top control rounds

Progression

  • Connect to sit out
