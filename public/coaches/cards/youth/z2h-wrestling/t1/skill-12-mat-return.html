Skill 12 — Mat Return

Tree: Top

Tier: Youth T1 — Recruit

Structure (A/B/C)

A — Secure Control

B — Lift with Balance

C — Return to Mat Control

Coaching Cues

  • “Lock before you lift”
  • “Stay balanced”
  • “Return under control”
  • “Flow back to pressure”

Common Mistakes

  • Lifting without control
  • Falling out of position
  • Losing control after return

Drill Ideas

  • Lift and return reps
  • Controlled return drills
  • Return to ride transitions

Progression

  • Connect to ride and breakdown sequences
