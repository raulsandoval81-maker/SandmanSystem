Skill 19 — Round Command

Journey: Adult R2G — Boxing Based

Tier: Adult T5

Category: Technique

Structure (A/B/C)

A — Control round pace, position, and decision-making

B — Manage risk, pressure, and scoring moments intelligently

C — Finish each round with tactical clarity and composure

Coaching Cues

  • “Command the round.”
  • “Control risk.”
  • “Win the moments.”
  • “Finish disciplined.”

Common Mistakes

  • Letting opponent dictate pace
  • Taking unnecessary risks
  • Ignoring round flow
  • Losing structure late

Drill Ideas

  • Score-aware sparring rounds
  • Round management scenarios
  • Last-minute command drills

Progression

  • Connect to Fight Command
  • Connect to System Application
