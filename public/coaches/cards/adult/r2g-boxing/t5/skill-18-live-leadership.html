Skill 18 — Live Leadership

Journey: Adult R2G — Boxing Based

Tier: Adult T5

Category: Technique

Structure (A/B/C)

A — Lead live rounds with composure, control, and discipline

B — Set the tone through pace, safety, and technical standards

C — Make partners better while maintaining competitive command

Coaching Cues

  • “Lead the round.”
  • “Set the standard.”
  • “Control the room.”
  • “Compete with discipline.”

Common Mistakes

  • Using leadership as ego
  • Going too hard without control
  • Ignoring partner safety
  • Losing composure when challenged

Drill Ideas

  • Leadership sparring rounds
  • Partner-control drills
  • Coach-observed live command rounds

Progression

  • Connect to Round Command
  • Connect to Coach Awareness
