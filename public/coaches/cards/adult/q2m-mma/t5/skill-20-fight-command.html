Skill 20 — Fight Command

Journey: Adult Q2M — MMA Based

Tier: Adult T5

Category: Technique

Structure (A/B/C)

A — Dictate where and how the fight happens

B — Control pace, space, pressure, and transitions

C — Recover command after chaos or resistance

Coaching Cues

  • “Command the fight.”
  • “Dictate the terms.”
  • “Stay composed.”
  • “Take control back.”

Common Mistakes

  • Letting opponent lead exchanges
  • Forcing control emotionally
  • Losing structure in chaos
  • Failing to reset after mistakes

Drill Ideas

  • Fight command rounds
  • Full-system control sparring
  • Coach scenario command drill

Progression

  • Connect to system application
  • Connect to legacy standard
