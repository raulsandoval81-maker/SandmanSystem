Skill 05 — Setup System

Journey: Adult Q2M — MMA Based

Tier: Adult T3

Category: Technique

Structure (A/B/C)

A — Create reactions through movement and feints

B — Build attacks from predictable reactions

C — Reset and rebuild the setup sequence

Coaching Cues

  • “Create the opening.”
  • “Build reactions.”
  • “Don’t force attacks.”
  • “Flow setup to finish.”

Common Mistakes

  • Attacking without setup
  • Repeating the same entry
  • Overcommitting feints
  • Standing still after setup

Drill Ideas

  • Feint-to-attack drill
  • Reaction setup rounds
  • Partner setup chain drill

Progression

  • Connect to weapon selection
  • Connect to live rounds
