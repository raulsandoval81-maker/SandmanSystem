Skill 12 — Return to Ride

Tree: Top

Tier: Youth T4 — Warrior

Structure (A/B/C)

A — Lift With Control

B — Return to Mat

C — Re-Establish Ride

Coaching Cues

  • “Stay tight on lift”
  • “Bring him back under control”
  • “Land ready to ride”

Common Mistakes

  • Losing control during return
  • Standing too tall
  • Not re-securing position

Drill Ideas

  • Return reps
  • Stand-up return drills
  • Top reset rounds

Progression

  • Connect to ride control loop
