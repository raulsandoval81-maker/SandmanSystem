Skill 24 — Return Control

Tree: Top

Tier: Teen T3 — Veteran

Structure (A/B/C)

A — Lift With Control

B — Return to Mat

C — Maintain Control

Coaching Cues

  • “Stay tight on lift”
  • “Return clean”
  • “Land in control”

Common Mistakes

  • Losing control mid-return
  • Standing too tall
  • Failing to re-secure position

Drill Ideas

  • Return to ride reps
  • Stand-up return drills
  • Continuous top control rounds

Progression

  • Connect to ride domination
