Skill 23 — Return to Ride

Tree: Top

Tier: Teen T2 — Champion

Structure (A/B/C)

A — Lift Control

B — Return to Mat

C — Re-Establish Ride

Coaching Cues

  • “Stay tight on the lift”
  • “Return clean, not wild”
  • “Land in control”

Common Mistakes

  • Losing control mid-return
  • Standing too tall
  • Failing to re-secure ride after return

Drill Ideas

  • Return + ride flow
  • Stand-up return reps
  • Continuous top control drill

Progression

  • Add turn chain
  • Connect to ride control system
