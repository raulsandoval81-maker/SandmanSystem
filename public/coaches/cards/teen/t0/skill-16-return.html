Skill 16 — Return

Tree: Top

Tier: Teen T0 — Apprentice

Structure (A/B/C)

A — Lift

B — Control

C — Return to Mat

Coaching Cues

  • “Lift with your legs”
  • “Stay tight to the body”
  • “Return with control”

Common Mistakes

  • Loose lift
  • Losing control mid-air
  • Slamming or losing position

Drill Ideas

  • Lift + return reps
  • Return from stand-up
  • Control on landing drill

Progression

  • Return to ride chain
  • Add breakdown after return
