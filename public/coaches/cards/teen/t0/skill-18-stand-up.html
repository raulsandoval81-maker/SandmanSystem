Skill 18 — Stand Up

Tree: Bottom

Tier: Teen T0 — Apprentice

Structure (A/B/C)

A — Build Up

B — Stand

C — Break Free

Coaching Cues

  • “Build, then stand”
  • “Hands first”
  • “Turn and face”

Common Mistakes

  • Standing without control
  • Leaving hands behind
  • Getting returned easily

Drill Ideas

  • Stand-up reps
  • Hand control break
  • Stand → return → repeat

Progression

  • Add escape chain
  • Add return defense
