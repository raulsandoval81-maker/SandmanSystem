Skill 09 — Advanced Jab

Journey: Teen P2L — Boxing

Tier: T3

Category: Technique

Structure (A/B/C)

A — Use multiple jab variations to control reactions

B — Blend timing, rhythm, and pressure through the jab

C — Build offensive systems from jab control

Coaching Cues

  • “The jab controls everything.”
  • “Different looks.”
  • “Touch, blind, pressure.”
  • “Build off the jab.”

Common Mistakes

  • Using the same jab every time
  • Throwing lazy lead hands
  • Ignoring footwork with the jab
  • Not recovering stance

Drill Ideas

  • Variable jab rounds
  • Pressure-jab sparring
  • Range-control jab drills

Progression

  • Connect to Setup System
  • Connect to Advanced Cross
