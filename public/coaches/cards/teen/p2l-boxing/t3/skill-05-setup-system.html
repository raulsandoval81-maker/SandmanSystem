Skill 05 — Setup System

Journey: Teen P2L — Boxing

Tier: T3

Category: Technique

Structure (A/B/C)

A — Build layered offensive setups before attacking

B — Use feints, pressure, and rhythm changes together

C — Create high-percentage attack opportunities

Coaching Cues

  • “Set it up first.”
  • “Create the reaction.”
  • “Build the opening.”
  • “Don’t force punches.”

Common Mistakes

  • Attacking without setup
  • Repeating the same entry
  • Telegraphing combinations
  • Overcommitting pressure

Drill Ideas

  • Feint-to-entry drills
  • Pressure setup rounds
  • Reaction-building sparring games

Progression

  • Connect to Pressure Flow
  • Connect to Offensive Layers
