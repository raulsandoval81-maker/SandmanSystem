Skill 09 — Jab System

Journey: Teen P2L — Boxing

Tier: T4

Category: Technique

Structure (A/B/C)

A — Use the jab to control pace, range, and reactions

B — Blend multiple jab variations into layered offense

C — Build complete exchanges through jab control

Coaching Cues

  • “The jab runs the system.”
  • “Different looks.”
  • “Control with the lead hand.”
  • “Build off the jab.”

Common Mistakes

  • Throwing lazy jabs
  • Using the same timing repeatedly
  • Ignoring footwork with the jab
  • Dropping the lead hand after punching

Drill Ideas

  • Variable jab rounds
  • Range-control sparring
  • Lead-hand reaction drills

Progression

  • Connect to Cross System
  • Connect to Hook System
